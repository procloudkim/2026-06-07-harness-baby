# ParentPick Guard Plan

## Objective

Ship a public-repo-ready Codex Plugin that helps newborn parents transform baby-product safety and trust source notes into a MECE report.

## Work Phases

1. Complete research gate and source governance.
2. Validate and update plugin manifest and marketplace.
3. Implement skill instructions and PowerShell report generator.
4. Add examples, templates, generated report, and screenshot asset.
5. Add README, final report, demo script, run log, and decisions.
6. Verify locally and record results.

## Current Method

Static local plugin with deterministic PowerShell tools. No web crawling, no paid APIs, no medical diagnosis.
